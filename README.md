# Hadooney — The Game

## What is Hadooney?
A mysterious creature collector set in a world where little blob-like beings called **Hadooneys** live in hidden pockets of the internet. You discover them, befriend them, and guide them through challenges.

## Why This Game?
- **Collecting** is satisfying (like Pokemon — but simpler)
- **Blobs are easy to draw** (circles with faces — no art skills needed!)
- **Browser-based** = works on MacBook + Android phone
- **Simple code** = Gowrish can learn by modifying it

## Game Loop (the core cycle that repeats)
```
Explore → Discover Hadooney → Befriend it → Train it → Challenge → Win → Unlock new area → Explore more
```

## Hadooney Types (starting set)
| Name | Color | Personality | Special Power |
|------|-------|-------------|---------------|
| Ember | 🔴 Orange-red | Bold, impatient | Fire burst — burns obstacles |
| Frost | 🔵 Ice blue | Calm, strategic | Freeze — stops enemies |
| Sprout | 🟢 Leaf green | Friendly, curious | Heal — restores energy |
| Zapp | 🟡 Electric yellow | Hyper, chaotic | Shock — stuns enemies |

## Controls
- Click/tap to move
- Click on a Hadooney to interact
- Simple turn-based challenges

## Tech Stack
- HTML5 Canvas (for drawing the game)
- JavaScript (for game logic)
- CSS (for menus and UI)
- GitHub Pages (for free hosting)
