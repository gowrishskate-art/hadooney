# Hadooney — Empire Wars

## Project
Browser-based empire conquest strategy game. Single-page HTML5 Canvas + JavaScript. Must work offline, on Android phone + MacBook.

## Game Requirements
- 20 eras (10 historical + 10 fictional), each with unique special powers
- Hex-based map, turn-based conquest
- Player picks era, picks 1-20 AI opponents
- Star rating per era (1-5 stars based on territory % conquered)
- Play time tracker (total hours/minutes/seconds across sessions)
- How to Play tutorial section
- localStorage for saving progress
- Touch + click support for mobile + desktop

## Era List
Historical: Ancient Egypt, Roman Empire, Viking Age, Medieval Europe, Ottoman Empire, Aztec Empire, Ming Dynasty, Edo Japan, British Empire, Industrial Revolution
Fictional: Cyber Dynasty, Quantum Realm, Star Empire, Dragon Dominion, Crystal Kingdom, Shadow Nexus, Thunder Legion, Void Collective, Ember Ascendancy, Frost Dominion

## Star Rating Thresholds
Star 1: 20%+, Star 2: 50%+, Star 3: 75%+, Star 4: 85%+, Star 5: 94%+ (based on territory %)

## Technical
- Single HTML file (index.html) with inline CSS + JS
- HTML5 Canvas for rendering
- No external dependencies
- Must be playable on phone (touch) and laptop (click/keyboard)
- GitHub Pages hosting
